## ✨ Features

### 🎙️ For Students
- **Audio Upload** — Upload MP3, WAV, M4A files (up to 100MB)
- **YouTube URL** — Paste any YouTube or video URL to process lectures
- **AI Transcription** — Accurate speech-to-text using OpenAI Whisper (beam search + temperature fallback)
- **AI Study Notes** — Structured, exam-ready notes generated by Google Gemini 2.5
- **Smart Quiz** — 5 AI-generated MCQ questions testing understanding, not just recall
- **Key Concepts** — Automatic extraction of keywords, bigrams, and definitions
- **Voice Energy Profile** — Visual chart showing teacher emphasis and speaking patterns
- **Teacher Emphasis** — Highlighted segments where the teacher spoke with emphasis
- **Timed Quizzes** — Configurable timer per question with visual feedback
- **Session History** — Access notes and scores from past live lectures
- **Live Lecture Mode** — Join a live session and get real-time transcription + notes

### 👨‍🏫 For Teachers
- **Live Lecture** — Start a session with a 4-digit code, stream microphone audio in real-time
- **Real-Time Processing** — Automatic transcription, concept extraction, and emphasis detection
- **Live Quiz** — Send quizzes to all students simultaneously during a lecture
- **Class Analytics** — View per-student performance, topic-wise strengths/weaknesses
- **Session History** — Review all past live sessions with notes and quiz results
- **PDF Export** — Download AI-generated notes as PDF

## 🚀 Quick Start

### Prerequisites
- **Python 3.9+**
- **FFmpeg** (auto-installed via `imageio-ffmpeg`)
- **Google Gemini API key** ([Get one free](https://aistudio.google.com/apikey))

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/ai-lecture-assistant.git
cd ai-lecture-assistant

# 2. Create virtual environment
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure environment
cp .env.example .env
# Edit .env and add your GEMINI_API_KEY

# 5. Run the application
python main.py
```

Open **http://localhost:5000** in your browser.

## ⚙️ Configuration

All configuration is done via environment variables (`.env` file):

| Variable | Default | Description |
|----------|---------|-------------|
| `GEMINI_API_KEY` | — | Google Gemini API key (required) |
| `GEMINI_MODEL` | `gemini-2.5-flash` | Gemini model to use |
| `WHISPER_MODEL` | `small` | Whisper model: `tiny`, `base`, `small`, `medium`, `large` |
| `SECRET_KEY` | auto | Flask session secret key |
| `HOST` | `127.0.0.1` | Server host |
| `PORT` | `5000` | Server port |
| `FLASK_DEBUG` | `true` | Debug mode |

### Whisper Model Comparison

| Model | RAM | Speed | Accuracy |
|-------|-----|-------|----------|
| `tiny` | ~1 GB | ⚡ Fastest | ★★☆☆☆ |
| `base` | ~1 GB | ⚡ Fast | ★★★☆☆ |
| `small` | ~2 GB | ⚡ Balanced | ★★★★☆ |
| `medium` | ~5 GB | 🐢 Slow | ★★★★★ |
| `large` | ~10 GB | 🐢 Slowest | ★★★★★ |
